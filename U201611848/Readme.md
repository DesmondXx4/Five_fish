# 陈田田的预习Demo——***场地租借***
## **程序功能**
1 记录显示租借者的身份信息    
2 根据不同身份和租用时间计算租金
## **程序开发工具**
Visual C++ 6.0                              
![](C:\Users\chenTT\Desktop\QQ图片20180813222504.png)
## **程序运行**
### ①代码组成及其功能
1 Time类：设置借出时间和归还时间   
2 Card类：打印租借信息   
3 Teacher类、Student类、Outschool类：card的派生类，计算租金
### ②运行过程
1 输入租借人员身份编号（学生、老师）或姓名（校外人员）及租借起始时间         
2 对应类工作，根据需要显示对应的租借发票
